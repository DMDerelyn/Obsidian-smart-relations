# Python Data Science Tools

A quick reference for the most commonly used Python libraries in data science and machine learning workflows.

## NumPy

The fundamental package for numerical computing. Provides n-dimensional arrays and mathematical functions.

## Pandas

Data manipulation and analysis library. DataFrames make tabular data easy to work with.

## Scikit-learn

Machine learning library with consistent API for classification, regression, clustering, and preprocessing.

## Matplotlib / Seaborn

Visualization libraries. Matplotlib is the foundation; Seaborn provides higher-level statistical plots.

## PyTorch / TensorFlow

Deep learning frameworks. PyTorch favored in research; TensorFlow popular in production.
